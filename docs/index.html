O OBSERV

O OBSERV é um observatório de dados com interpretação de dados, geração de indicadores e gráficos.
Tudo isso para apoiar a visualização dos indicadores dos dados da sua pesquisa.

Visualize os resultados de pesquisas dos vários projetos cadastrados.

Serviços

Observatórios

Crie observatórios à partir dos mais variados dados de projetos de pesquisa. Visualize os resultados de pesquisas dos diferentes projetos cadastrados.

Projetos

Crie e compartilhe os dados da sua pesquisa. Escolha os dados que deseja disponibilizar, interprete as informações, gere indicadores e gráficos, crie e gerencie os seus observatórios.

Colaboradores

Trabalhe em equipe, interaja com os dados das pesquisas, gere seus próprios indicadores e contribua com o conhecimento do projeto.

lock_open

Dados Abertos

Explore nossa base dados. O OBSERV disponibiliza dados de pesquisas de projetos do NMT (Núcleo de Medicina Tropical) sobre doenças tropicais.

Sobre

O OBSERV é um produto desenvolvido pelo projeto TropicalHazards-BI, pertencente à disciplina de Engenharia de Produto de Software em conjunto com a disciplina de Métodos de Desenvolvimento de Software, que visa desenvolver um sistema de Observatório de Dados, inicialmente direcionado para pesquisadores e profissionais de saúde do Núcleo de Medicina Tropical (NMT) da Universidade de Brasília (UNB), para atender a necessidade de visualização de dados dos seus diferentes projetos em uma mesma plataforma.

2018.1-TropicalHazards-BI is maintained by fga-gpp-mds .
